# 启发式 v2 评分规则参考

> `classify.py` 内置这套规则；本文档是人类可读版，改规则时同步更新脚本。

## 文件名评分

| 关键词 | 得分 | 说明 |
|---|---|---|
| `张雪峰：` 前缀 | -3 | 独白类强信号 |
| `家长/孩子/闺女/儿子/姑娘/女儿` | +2 | 只算一次最高 |
| `分/怎么选/适合`（且无独白前缀） | +1 | 只算一次 |
| `段子/耍帅/好男人/价值三百` | -2 | 娱乐段子 |

## 正文评分（前 500 字）

| 命中 | 得分 | 说明 |
|---|---|---|
| `老师我想问/我们家/我闺女/我儿子/我多少分` | +1 | 家长主动发问 |
| `今天给大家/各位家长/在直播间/很多人会问/如果有家长问` | -1 | 独白起手 |

## 列表结构识别（前 2000 字）

任一 pattern 命中 ≥2 次即判定含列表结构：
- `第[一二三四五六七八九十]`
- `[一二三四五六七八九]个建议`
- `[一二三四五六七八九]个误区`
- `[一二三四五六七八九]问`

## content_type 判定

| 条件 | content_type |
|---|---|
| 总分 ≥ +2 | 对话 |
| 总分 ≤ -2 且含列表结构 | 专题 |
| 总分 ≤ -2 且无列表结构 | 独白 |
| 其他 | 其他 |

## segment_type

- `片段`：字符数 ≤ 8000
- `整场`：> 8000（打标但搁置，等 zxf-segment skill）

## signal_hint（仅对话类）

| 条件 | signal_hint |
|---|---|
| 文件名含三位数分数 `\d{3}`，或 `强基/民办/复读/选科/考研` | high_candidate |
| 文件名含 `情绪/焦虑/吐槽/崩溃/抑郁暗恋` | low_candidate |
| 其他对话 | medium_candidate |
| 非对话 | N/A |

## 误报率目标

- v2 目标 ≤10%（实测已达标）
- 如果误报率回升，先看是否有新的文件名模板未纳入正负词表
